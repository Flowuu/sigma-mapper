# Sigma-Mapper
Sigma-Mapper is a simple manual map and `LoadLibraryA` injector for 64-bit process/game.

## Features
- Parses PE headers and sections
- Maps import address
- Handles relocations

## Usage
1. Compile
2. Select process, DLL and method

## Requirements
- Windows (x64)
- Administrator privileges
